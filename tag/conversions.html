    Conversions

    Conversions

    Every attribute has a conversion function, allowing attributes to take in rich input types, but keep the variance low for the backends. The conversions are overloadable as explained in Extending, making it simple to integrate custom types.

    MakiE.to_color — Function.
    to_color(b, x)

    Colors.Colorants

    source

    A Symbol naming a color, e.g. :black

    source

    A String naming a color, e.g. :black or html style #rrggbb

    source

    A Tuple or Array with elements that to_color accepts. If Array is a Matrix it will get interpreted as an Image

    source

    Tuple{<: ColorLike, <: AbstractFloat} for a transparent color

    source

    source

    A Symbol/String naming the gradient. For more on what names are available please see: `available_gradients()
